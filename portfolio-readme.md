Simple Portfolio

This is a minimal static portfolio site.

Files created:
- index.html — main page
- styles.css — styling
- script.js — small interactions (mobile menu, smooth scroll)

Quick start:
1. Open `index.html` in your browser.
2. Edit the text in `index.html` to add your name, bio, and project links.
